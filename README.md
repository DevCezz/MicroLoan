MicroLoan
=========

**Projekt powstał z powodu:**

- procesu rekrutacyjnego na stanowisko Java Developer

Projekt ma na celu obsługę wydawania pożyczek oraz ich przedłużania. Przy wykonaniu tych operacji sprawdzane są
warunki, które weryfikują czy klient może daną pożyczkę dostać albo odłożyć. Wszystko odbywa się w bazie danych H2.
